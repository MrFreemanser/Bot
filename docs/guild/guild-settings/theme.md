# Theme

## ;theme

Configure theme settings.

### ;theme color

Displays current theme color of the server.

```yaml
Aliases:
- colour

Usage:
;theme color
```

#### ;theme color set

Set a new theme color for server.

```yaml
Usage:
;theme color set <color>
```

#### ;theme color remove

Removes the theme color of server which was set previously.

```yaml
Aliases:
- delete

Usage:
;theme color remove
```
