# Leaderboards

## ;leaderboards

Displays top members with maximum chat experience points.

```yaml
Aliases:
- leaderboard
- lb

Usage:
;leaderboards
```

### ;leaderboards global

Displays top users with maximum chat experience earned globally across all servers.

```yaml
Aliases:
- globals
- cosmos
```
